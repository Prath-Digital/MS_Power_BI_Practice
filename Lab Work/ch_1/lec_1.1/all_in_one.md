# Power BI Questions and Answers

## Q1
**Question:** Download and install Power BI Desktop from the Microsoft Store or the official Microsoft website. Explore and document settings like:
- Language
- Preview features
- Auto-recovery

**Answer:** To install Power BI Desktop, download it from the Microsoft Store or the official Microsoft website. After installation, explore settings under Options and Settings: adjust Language for regional preferences, enable Preview features for new functionalities, and configure Auto-recovery to safeguard your work.

## Q2

**Answer:** Open Power BI Desktop, navigate through the interface, and take screenshots. Label key components such as the Ribbon (for tools and commands), Data Pane (for data management), and Fields Pane (for field selection).

## Q3
**Question:** Write a short note on the differences between:
- Power BI Desktop
- Power BI Service
- Power BI Mobile

**Answer:** Power BI Desktop is a Windows application for creating and editing reports locally. Power BI Service is a cloud-based platform for sharing and collaborating on reports. Power BI Mobile offers on-the-go access to reports via mobile devices.

## Q4
**Question:** Research and list five industries that use Power BI. For each, describe one real-world application (e.g., healthcare, retail, finance).

**Answer:** 
- **Healthcare:** Used to track patient outcomes and optimize hospital resource allocation.
- **Retail:** Analyzes sales data to improve inventory management.
- **Finance:** Monitors financial performance and predicts market trends.
- **Manufacturing:** Tracks production efficiency and reduces downtime.
- **Education:** Analyzes student performance to enhance teaching strategies.

## Q5
**Question:** Write a brief report on the responsibilities of a Data Analyst using Power BI.

**Answer:** A Data Analyst using Power BI is responsible for collecting and cleaning data, creating insightful visualizations, generating reports, and providing actionable insights to support decision-making processes.

## Q6
**Question:** List and explain 3 key skills needed for a Data Analyst working in Power BI.

**Answer:** 
- **Data Visualization:** Ability to create clear and effective visual representations of data using Power BI.
- **Data Analysis:** Skill to interpret complex datasets and derive meaningful insights.
- **Technical Proficiency:** Knowledge of Power BI tools, DAX, and data modeling techniques.

## Q8
**Question:** Write a short scenario:
- Imagine you are a Business Analyst and your colleague is a Data Analyst. Define how both of you will work on a retail sales report. What will each of you focus on?

**Answer:** As a Business Analyst, I’ll gather requirements and define the report’s purpose, while my colleague, the Data Analyst, will focus on analyzing sales data and creating visualizations in Power BI to highlight trends.

- Required skills
- Responsibilities
- Tools

## Q10
**Question:** Write a one-line purpose statement for each Power BI component.

**Answer:** 
- **Ribbon:** Provides access to tools and commands for report creation.
- **Data Pane:** Manages and organizes data sources and tables.
- **Fields Pane:** Allows selection and manipulation of data fields.

## Q12
**Question:** List 3 advantages and 3 limitations of Power BI over Tableau.

**Answer:** 
- **Advantages:** Cost-effective with a free version, strong Microsoft integration, easy learning curve.
- **Limitations:** Limited advanced visualizations, less flexible with non-Microsoft tools, performance issues with very large datasets.
